## Troubleshooting & FAQ

**Q: How do I update Crystal PM?**
A: Updates are managed through the Crystal PM update mechanism. Ensure all workstations are synchronized to the same version. Contact support if you encounter version mismatch issues.

**Q: Scanner not visible in Crystal PM?**
A: Check scanner drivers, USB connection, and Crystal PM scanner configuration settings. Restart the application if needed. Contact support for persistent issues.

**Q: How do I optimize Crystal PM performance?**
A: Ensure adequate system resources, close unnecessary applications, check network connectivity (for cloud-hosted), and keep the software updated to the latest version.

**Q: How do I update VSP catalog/bill codes?**
A: Navigate to the VSP section in Crystal PM, download the latest catalog update, and follow the bill code update procedure in the help documentation.

**Q: How do I install Crystal PM on a new workstation?**
A: Follow the workstation installation guide in the help center. Ensure the new workstation meets system requirements and can connect to the Crystal PM server/cloud instance.

**Q: How do I back up the database?**
A: For on-premise installations, follow the database backup procedure in the help center. Cloud-hosted installations have automated daily encrypted backups.

**Q: What are the FTC eyeglass prescription rules?**
A: The FTC Eyeglass Rule requires that prescribers provide patients with a copy of their eyeglass prescription immediately after the eye exam, at no additional cost. Crystal PM supports compliance with prescription printing and documentation.

**Q: How do I handle a claim denial?**
A: Review the denial reason in the Claims Report, correct any errors in the claim, and resubmit using the claim resubmission feature. For complex denials, contact the insurance carrier or use Crystal PM's RCM service.

**TWAIN Scanners (HP, Brother, Epson, Xerox, Canon, Ambir, etc.):**
1. Install the full manufacturer software package first (must include TWAIN drivers — WIA drivers won't work)
2. For multifunction devices (print/scan/fax), install the complete software suite, not just the basic driver
3. In Crystal PM: Configuration > Scanner Options > Scanner Twain or Twain 2

1. Install ScanSnap Home software from Ricoh's website

**Troubleshooting scanners:**

## Insurance Claim Denial Guide

### Understanding Denial Code Prefixes
- **CO (Contractual Obligations):** Write-off amounts per provider-payer agreement — do NOT bill to patient
- **PR (Patient Responsibility):** Balance the patient legally owes (deductible, coinsurance, copay) — collect from patient
- **OA (Other Adjustment):** Adjustments that don't fit CO or PR categories
- **PI (Payer Initiated):** Reductions initiated by the payer

### Common Optometry Denial Codes

**CO-4 (Procedure code inconsistent with modifier):**
- Check that modifiers match the procedure code
- Common with -25 modifier on E/M codes
- Verify laterality modifiers (RT/LT) are correct

**CO-16 (Missing or invalid information):**
- Check Remark Codes (RARC) for specific missing field
- Common causes: incomplete patient info, wrong diagnosis codes, missing authorization
- Fix: verify all demographics, ensure prior auth is on file

**CO-97 (Service already included/bundled):**
- The service is bundled with another already-paid procedure
- Review bundling rules — do not separately bill bundled services
- May need modifier -59 (Distinct Procedural Service) if services are truly separate

**PR-96 (Non-covered charge):**
- Service not covered under patient's plan
- For Medicare: must have signed ABN (Advance Beneficiary Notice) BEFORE service to bill patient
- Without ABN, practice cannot collect from patient regardless of PR-96

**CO-18 (Duplicate claim):**
- Check if original claim is still processing before resubmitting
- Wait 30 days, then verify with payer before refiling

**CO-29 (Time limit for filing has expired):**
- Each payer has filing deadlines (typically 90 days to 1 year)
- Track claim submissions and follow up promptly
- Late submissions are automatically denied with no appeal in many cases

### Top 6 Eye Care Claim Rejection Causes

1. **Missing/invalid patient information** — Transposed ID digits, wrong DOB, outdated address. Fix: verify demographics at every check-in, use digital intake forms
2. **Duplicate claims** — Claim submitted twice for same service date. Fix: check claim status before refiling
3. **Missing/invalid codes or modifiers** — Wrong CPT code or missing required modifier. Fix: follow Local Coverage Determinations, keep coding current
4. **Patient not eligible** — Coverage inactive at time of service. Fix: verify eligibility and policy dates before exam
5. **Missing/invalid provider NPI or TIN** — Billing provider identifiers incorrect. Fix: confirm NPI and TIN are correct and on file with payer
6. **Provider not credentialed** — Provider not on insurance panel. Fix: verify credentialing status, track re-credentialing deadlines

### Claim Denial Workflow in Crystal PM
1. Run the Claims Report to identify denied claims
2. Note the denial code and remark codes
3. Correct the issue (patient info, codes, modifiers, etc.)
4. Use "How To Resubmit Correct Claims" feature to refile
5. Track resubmission in Claims Report
6. For persistent denials, contact the payer or escalate to Crystal PM RCM team

## Performance & Speed Issues

This is one of the most commonly reported problems in reviews and the forum. Crystal PM can slow down significantly when multiple staff members are logged in simultaneously.

### Quick Fixes for Slowness

1. **Reboot your server and all affected workstations** -- this is the first thing support will tell you to do.

2. **Disable CPM Messaging** (if you don't use it):
   - Go to **Admin > Company > General**
   - Check **"Turn Off All Messaging"**

3. **Disable Audit Log** (if you are NOT participating in Meaningful Use or MIPS):
   - Go to **Admin > Company > Security 2**
   - Check **"Disable Audit Log"**

4. **Optimize the Database**:
   - Go to **Admin > Computers > "Optimize Database"**
   - **WARNING**: Crystal will appear frozen/unresponsive during this process -- this is normal. Do NOT force-close it. Wait for it to complete.

5. **Post-update lag**: After Crystal PM updates, the software may be sluggish. A server reboot after updates often resolves this.

6. **Force updates on client machines**: If a workstation didn't receive an update, go to **Configuration** to force close CrystalPM and re-run the update.

### Anti-Virus Note
AVG, Norton, and McAfee are **NOT recommended** with Crystal PM. They can cause significant performance degradation. Use a different AV product.

## Scheduling Tips

### Color-Coded Appointments
Crystal PM's schedule is color-coded by appointment type and status. The color lists for appointment types and appointment status are **linked** -- the nth item on one list shares the same color as the nth item on the other list. Keep this in mind when setting up your appointment types and statuses.

### Setup Steps
- Create appointment types in **Admin > Schedule** section
- Create rooms
- Set up schedule templates with defaults

### Multi-Doctor Scheduling
Crystal PM supports multi-doctor scheduling with:
- Customizable appointment types per doctor
- Drag-and-drop appointment management
- Trackable room assignments
- Wait/reschedule list features
- Automated re-scheduling for canceled/missed appointments

- **Outside-hours warning**: Crystal will prompt when scheduling appointments outside an employee's set hours (v6.4.16+, can be disabled in v6.4.27)
- **Online scheduling integration**: Schedule Your Exam allows patients to book online, and appointments automatically import into Crystal PM's schedule.

## Database Backup & Maintenance

### Backup Best Practices
- **Back up daily** -- this is one of the most important processes you should perform
- Schedule backups between **1:00 AM and 5:00 AM** when no one is working
- Security must be set to **Level 1** in Admin > Company > Security for the backup option to appear on the server

### Crystal Cloud Backup
- Secure "air-gapped" daily backup -- disconnected from the network after each backup
- Hackers cannot access the backup even during a cyber attack
- **My Data My Way** service creates daily backups synced to your practice location
- Data can be downloaded at any time for peace of mind

### Database Optimization
Run **Admin > Computers > "Optimize Database"** periodically to maintain performance. The application will freeze temporarily during this process.

### Updating Crystal PM
- Go to **Admin > Updates** and click **"Check For Updates Online"**
- If a new version is available, click **"Update"**
- Reboot the server after updates
- Client machines that didn't update can be forced via the Configuration menu

### Critical Bugs to Be Aware Of
